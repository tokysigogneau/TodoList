# Todo List 

### Description 
Todo List is an app that allow user to create tasks, and manage/edit them.
A task consist of : 
- name
- date
- description
- progression("to do" by default)

The user can display all the tasks he created on page my_tasks.html.
Then he can edit the task by clicking on the name of the task wich will lead to another page edit_task.html.
Or he can change the progression directly on the task list on my_task.html without having to move to edit page.
The user can also delete unwanted tasks on the task list at my_task.html.
